## Conference

Wey Wey Web is a tech conference about User Interfaces.

From software development to creative design, including UX and product design. A 2 days of fun to mingle with interface lovers.

## Technical Talks

Wey Wey Web is all about different technologies, best practices, and methodologies related to Web development. No matter if you are into Javascript, PHP, Python, WASM, Java or C#. If you are dealing with the User Interface, then THIS IS YOUR PLACE!

## Topics

- Web programming
- JS frameworks
- Web design UX/UI
- Product design
- Mobile development

This is the first edition of the event and it will take place in Malaga on the 15th and 16th of December.

It is organised by the same team as <a target="_blank" href="https://www.jonthebeach.com/">J On The Beach</a>.
