## The Event

**Whaaaat the WEB!**

In only 2 days you will be able to check the pulse of the latest trends with regards to **Web Technologies**. Below are the main topics that will be covered in **Wey Wey Web**.

- Web Programming
- JS & Other Languages
- UX/UI Design
- Product Design
- Software Engineering
- Accessibility & Web Performance

## Dev/Design Talks

Wey Wey Web is the ultimate fusion of design and code. From cutting-edge front-end technologies to the freshest UX and product design practices, we've got it all covered. Explore everything from multi-modal interfaces to the latest web trends, diving deep into both pure coding and the visual aspects.

No matter if you are a **Javascripter**, a **Figma-maniac**, a **PHP lover**, a **Rive fan**, or a **Pythonist**, if you are dealing with the User Interface, then THIS IS YOUR PLACE!

## About us

Wey Wey Web is organised by **Yay-Yay Events** with the support of the fem-tech association **Yes We Tech**. This is a non-profit event meaning that all funds raised will be spent to cover the expenses.

This event is committed to social inclusion, and we actively encourage underrepresented groups to take part in this conference as attendees as well as speakers.

## What to expect for 2024

- **20 Speakers**: International speakers from all around the world sharing their best practices.
- **3 Keynotes**: World class creators of technologies will open and close the event.
- **420 Attendees**: Mingle with top creators, promising developers, and UI professionals.
- **2 Days of fun**: YAAAAAAAY!
