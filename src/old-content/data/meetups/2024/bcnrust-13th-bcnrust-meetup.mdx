> ¡Tenemos nuevo evento con dos charlas!

## Introduction to programming Microcontrollers with Rust by [Andreas Bergmeier](https://fosstodon.org/@abergmeier)

With Rust getting more and more support for embedded Architectures, it becomes a viable option for use with Microcontrollers. This talk will try to give an overview about the Rust support landscape in this area as well as some real world examples using ESP\* devices.

## Desvelando la estructura de memoria de los tipos de datos en Rust por [Germán Küber](https://www.linkedin.com/in/germankuber/)

Esta charla se sumerge en el fascinante proceso de ejecución de un binario y cómo se mapean los segmentos en la memoria, explorando el propósito y funcionamiento de la memoria stack y heap. Profundizaremos en cómo se organizan en la memoria los valores de los distintos tipos de datos de Rust, incluyendo enteros, char, Vector, slice, String, string slice, estructuras, enumeraciones y punteros inteligentes como Box, Rc, Arc, además de objetos Trait y rasgos de funciones como FnOnce, FnMut y Fn.
