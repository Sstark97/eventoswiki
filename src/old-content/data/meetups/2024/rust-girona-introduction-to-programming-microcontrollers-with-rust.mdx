## Details

With Rust getting more and more support for embedded architectures, it becomes a viable option for use with Microcontrollers.

In this talk, Andreas Bergmeier will overview the Rust support landscape in this area and some real-world examples using ESP\* devices.

This time we have the pleasure of having an international speaker, Andreas Bergmeier [@abergmeier](https://fosstodon.org/@abergmeier).

The event will be exclusively in English and we'll broadcast and record it.
