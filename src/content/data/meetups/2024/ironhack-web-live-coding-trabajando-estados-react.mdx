## Detalles

En este evento de live coding veremos como los estados de los componentes de React son una parte fundamental en el ecosistema reactivo que nos rodea. Durante el evento crearemos, desde cero, diez ejemplos de uso de estados en React a través de los que aprender las claves para controlar el aspecto y comportamiento de los componentes funcionales. Desde los casos más comunes hasta ejemplos disruptivos, podrás revisar y experimentar con los códigos que desarrollaremos durante el evento y te enviaremos después.

Si tienes alguna duda o se acaban las entradas, escribe a contact.spain@ironhack.com

📅 ¿Cuándo?: Miércoles, 05 de junio a las 19:00h.

🕐 Duración: 2h.

👥 Modalidad: Presencial en el campus de Madrid

🧑¿Quién lo imparte?: Germán Álvarez, Senior Coding Instructor

👩‍💻 ¿Es para ti este taller?: Si te interesa el desarrollo web y la tecnología y crees que es vital para el futuro, ¡no te lo puedes perder! No necesitas ningún tipo de conocimiento previo, ¡solo ganas de aprender!

Tras finalizar el taller, disfrutaremos de un espacio de networking donde podrás hablar con el equipo de Ironhack acompañados de cervezas.
